# docs/

Project documentation organized by audience.

| Subdir           | Audience              | Contents                                       |
|------------------|-----------------------|------------------------------------------------|
| `adr/`           | Maintainers           | Architecture Decision Records (immutable log). |
| `architecture/`  | Contributors          | System diagrams, data model, sequence flows.   |
| `operations/`    | Operators / on-call   | Runbook, alarms, restore drills.               |
| `security/`      | Security reviewers    | Threat model, SECURITY.md, reporting process.  |
| `contributors/`  | New contributors      | Environment setup, testing, release process.   |
