# packages/contracts

Canonical API contract: the FastAPI app exports the OpenAPI spec to
`openapi.json` (pinned to version `1.0.0` per `/make-docs`); the frontend
regenerates `frontend/src/api/schema.d.ts` from it via `openapi-typescript`.

## Files

- `openapi.json` — generated by `backend/scripts/export_openapi.py`. Do not
  hand-edit; changes come from FastAPI route signatures + Pydantic models.
- `src/providers.ts` — shared TypeScript interface declarations that mirror
  the `MailboxProvider` / `LLMProvider` / `AuthProvider` Python protocols
  (plan §19.6). Kept by hand for now; a codegen step lands in a later phase.

## Regenerating

```bash
make docs
```

CI fails the PR if any generated artifact drifts from the committed state.
